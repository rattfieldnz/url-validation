Welcome to the Url Validation project

This is a Laravel composer package for validating URL's.

Installation (Composer)

The easiest way to install this packages is via composer. To do this, ensure you have a composer.json file in your project root, then navigate to the project root and enter the following command:

          
  cd your_project_root
  composer require "rattfieldnz/url-validation": "dev-master"

This project is still in initial development, hence the 'dev-master' requirement. Once thorough testing has been done, the requirement will be changed to 'master'.

Validation for this package will include:

  • Validating for correct syntax, specified by internal regex libraries
  • Validating whether a given url exists or not
  • Checking if a given url exists on spam-related databases
